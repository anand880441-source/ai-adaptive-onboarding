# AI-Adaptive Onboarding Engine

An intelligent system that transforms corporate onboarding from static, one-size-fits-all curricula into personalized, adaptive learning pathways based on individual skill gaps.

## 🚀 Quick Start

### Prerequisites
- Docker and Docker Compose (recommended)
- Python 3.9+
- Node.js 14+ (for local development)

### Installation & Setup

#### Using Docker (Recommended)
```bash
git clone https://github.com/yourusername/ai-adaptive-onboarding.git
cd ai-adaptive-onboarding
docker-compose up --build
